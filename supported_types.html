FSharp.Json

Supported Types

Here's full list of F# types that are supported by FSharp.Json library.

int

Represented as number in JSON.

int64

Represented as number in JSON.

float

Represented as number in JSON.

decimal

Represented as number in JSON.

string

Represented as string in JSON.

bool

Represented as bool in JSON.

DateTime

Represented as string according to ISO 8601. Could be represented as number epoch time using transform.

DateTimeOffset

Represented as string according to ISO 8601. Could be represented as number epoch time using transform.

Guid

Represented as string.

Enum

Represented as string that is enum member name. Could be represented as enum member id. Read more on Enums page.

option

Option is not represented by itself. None value might be represented as null or omitted. Read more on Null Safety page.

tuple

Tuples are represented as list in JSON.

record

Records are represented as object in JSON.

map

Map is represented as object in JSON.

array

Array is represented as list in JSON.

list

List is represented as list in JSON.

union

Unions are represented as object with special structure in JSON. Read more on Unions page.
